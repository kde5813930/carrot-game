# 🥕carrot-game
vanillaJS를 활용하여 간단한 게임 구현



## 게임 기능
play 버튼을 클릭하면 랜덤을 배치된 🥕과 🐛가 배치되어<br>
타임워치가 작동하여 해당 시간내 당근만을 클릭해 제거하는 게임입니다.<br>
성공하면 'win'이라는 성공 팝업창이 뜨고, 시간내에 성공하지 못하거나 벌레를 클릭했을 때 'lose' 팝업창이 뜹니다.<br>
리셋 버튼을 클릭하면 새로 게임을 시작할 수 있습니다.

### 활용 기술 
- HTML
- CSS
- vanillaJs
